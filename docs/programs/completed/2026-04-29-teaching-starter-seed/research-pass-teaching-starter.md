# Research Pass: Teaching Starter

The starter needs to teach durable agent workflow primitives without requiring access to any private project.

The key concepts are:

- `AGENTS.md` gives agents local rules and routing.
- `README.md` orients humans and explains the repo.
- `docs/programs/` holds initiative-level memory.
- `docs/exec-plans/` holds slice-level execution contracts.
- `docs/prompts/` holds reusable copy-paste workflow prompts.
- `.agents/skills/` holds repeatable skill wrappers.

The starter should include contracts, templates, prompts, skills, and one completed example so learners can copy the pattern into their own project.
