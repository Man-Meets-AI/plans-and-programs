# Planning Brief 1

Create a public educational starter repo that teaches Programs, ExecPlans, prompts, skills, and AGENTS routing.

The implementation should:

- be project-agnostic
- include root onboarding
- include Program and ExecPlan contracts
- include templates
- include prompt assets
- include skill wrappers
- include one completed Program and child ExecPlan
- include a simple validation command

The work is complete when a learner can clone or fork the repo, read the README, inspect the completed example, copy the templates, and run validation without installing dependencies.
