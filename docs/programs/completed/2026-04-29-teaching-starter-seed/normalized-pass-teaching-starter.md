# Normalized Pass: Teaching Starter

The public starter should optimize for immediate adoption.

It should not teach from private repos or require a runtime service. It should present the minimal artifact set that makes Programs and ExecPlans useful:

- a root orientation README
- a compact root AGENTS file
- Program contract and template
- ExecPlan contract and template
- prompt assets for writer, griller, execute, validate, and Program refresh
- shared skill wrappers that point to those prompts
- a no-dependency validator
- one completed seed Program and child ExecPlan
