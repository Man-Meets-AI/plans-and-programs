# ExecPlan Execute

Execute one approved ExecPlan while keeping it current.

## Use When

- the plan exists
- the plan has been reviewed or grilled
- the user wants implementation, not more planning

## Procedure

1. Read `AGENTS.md`.
2. Read `docs/exec-plans/PLANS.md`.
3. Read the target ExecPlan.
4. Identify the active milestone.
5. Implement the next smallest validatable slice.
6. Run focused validation.
7. Update the ExecPlan with progress, discoveries, decisions, and proof.
8. Repeat until complete or blocked.
